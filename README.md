## 🛠️기술 스택
🧩 Frontend
- Vue.js 3 
- Vue Router
- Pinia

🎨 UI & UX
- CSS3 
- Chart.js

📦 Backend 
- JSON Server 
- Axios 
- Glitch

🛠 협업 & 버전 관리
- Git / GitHub
- Vercel
- Notion / Figma

## 주요 기능

- 캘린더 기반 수입/지출 기록
- 카테고리별 소비 통계 시각화
- 날짜별 메모 기능
- 로그인 및 마이페이지 관리
